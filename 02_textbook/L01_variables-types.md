# L01 変数とデータ型 — プログラムの「箱」

> **対象レベル:** プログラミング未経験者  
> **所要時間:** 90〜120分  
> **到達目標:** 変数の宣言・代入・printf/scanf が自力で書ける

---

## 1. コンセプト概要

**変数（variable）** とは、プログラムがデータを一時的に記憶しておくための「名前付きの箱」です。プログラムは実行中に数値・文字・文字列など様々なデータを扱いますが、それらを保存する場所が必要です。C言語では変数を使ってコンピュータのメモリ（RAM）上に領域を確保し、名前をつけてアクセスします。変数には「型」があり、何を入れる箱なのかをコンパイラに教えます。型によって箱の大きさ（バイト数）と入れられる値の範囲が決まります。

---

## 2. なぜ必要か

電卓アプリを想像してください。「12 + 7」を計算して「19」と表示するには、入力した12と7を一時的にどこかに覚えておかなければなりません。その「一時的に覚えておく場所」が変数です。

変数がなければ：
- 計算の途中結果を保存できない
- ユーザーの入力を受け取れない
- 複雑な処理（ループ・条件分岐）ができない

プログラムのあらゆる動作は、何らかのデータを読み書きする操作です。変数はその基本単位です。

---

## 3. 現実世界のアナロジー

**ロッカー（貸し出し荷物棚）のイメージ**

```
  ┌─────────────────────────────────────────┐
  │  ロッカーNo.101  ← メモリアドレス        │
  │  ラベル: apples  ← 変数名               │
  │  中身:    5      ← 値（value）           │
  │  サイズ: int型   ← データ型              │
  └─────────────────────────────────────────┘
```

図書館のロッカーと同じです：
- **ロッカー番号**（101）= メモリアドレス（コンパイラが自動管理）
- **ラベル**（apples）= 変数名（プログラマが命名）
- **中身**（5）= 現在の値
- **ロッカーのサイズ**（大・中・小）= データ型（int, double, char）

「apples が入ったロッカーを開けて」と言えば中身の5が取り出せる。これが変数アクセスです。

## 4. 構文説明

### 4.1 変数の宣言（箱を作る）

```
  データ型  変数名  ;
  ────┬────  ──┬──
      │        └── 好きな名前（英字始まり、英数字とアンダースコア）
      └── int / double / char など
```

```c
int apples;        // 整数用の箱「apples」を作る
double price;      // 小数用の箱「price」を作る
char grade;        // 文字用の箱「grade」を作る
```

### 4.2 変数への代入（箱に値を入れる）

```c
apples = 5;        // apples の箱に 5 を入れる
price  = 120.5;    // price の箱に 120.5 を入れる
grade  = 'A';      // grade の箱に文字 'A' を入れる
```

### 4.3 宣言と初期化を同時に（推奨）

```c
int    apples = 5;
double price  = 120.5;
char   grade  = 'A';
```

### 4.4 主なデータ型

```
┌──────────┬──────────┬──────────────────────────┬──────────────────┐
│ 型名     │ バイト   │ 値の範囲                 │ 使い方           │
├──────────┼──────────┼──────────────────────────┼──────────────────┤
│ int      │ 4 bytes  │ -2,147,483,648 〜         │ 整数全般         │
│          │          │  2,147,483,647            │ (個数・点数など) │
├──────────┼──────────┼──────────────────────────┼──────────────────┤
│ double   │ 8 bytes  │ ±1.7×10^308 (倍精度)     │ 小数を含む実数   │
│          │          │ 有効桁: 約15桁            │ (価格・距離など) │
├──────────┼──────────┼──────────────────────────┼──────────────────┤
│ char     │ 1 byte   │ 0 〜  255 (ASCII)         │ 1文字            │
│          │          │                           │ ('A','0','!' 等) │
├──────────┼──────────┼──────────────────────────┼──────────────────┤
│ float    │ 4 bytes  │ 単精度浮動小数点           │ メモリ節約時     │
│          │          │ 有効桁: 約7桁             │ (通常は double)  │
└──────────┴──────────┴──────────────────────────┴──────────────────┘
```

### 4.5 printf の書式指定子

```
printf("文字列 %書式指定子 文字列\n", 変数名);
          └──────────┬──────────┘
                     └── 変数の値がここに埋め込まれる
```

```
┌────────┬────────────┬────────────────────────────────┐
│ 指定子 │ 対応する型 │ 例                              │
├────────┼────────────┼────────────────────────────────┤
│  %d    │ int        │ printf("%d", 42)  → 42          │
│  %f    │ double     │ printf("%f", 3.14)→ 3.140000    │
│ %.2f   │ double     │ printf("%.2f",3.14)→ 3.14       │
│  %lf   │ double     │ scanf に使う (printf では %f でも可) │
│  %c    │ char       │ printf("%c", 'A') → A           │
│  %s    │ 文字列     │ printf("%s", "hello") → hello   │
└────────┴────────────┴────────────────────────────────┘
```

### 4.6 scanf の使い方（ユーザー入力を受け取る）

```c
int n;
scanf("%d", &n);   // &n = 「n のアドレス（場所）」を渡す
                   //  & を忘れると動作が未定義になる！
```

`&` はアドレス演算子。「変数 n の住所」をコンピュータに教えるために必要です。

## 5. ステップ実行トレース

次のプログラムを1行ずつ追います。

```c
#include <stdio.h>
int main() {
    int    apples = 5;
    double price  = 120.5;
    char   grade  = 'A';
    printf("りんご: %d個\n", apples);
    printf("値段: %.1f円\n", price);
    printf("成績: %c\n", grade);
    return 0;
}
```

**実行トレース表**

| 行 | 実行内容 | apples | price | grade | 画面出力 |
|---|---|---|---|---|---|
| 3 | `int apples = 5;` | **5** | (未作成) | (未作成) | — |
| 4 | `double price = 120.5;` | 5 | **120.5** | (未作成) | — |
| 5 | `char grade = 'A';` | 5 | 120.5 | **'A'** | — |
| 6 | `printf("りんご: %d個\n", apples);` | 5 | 120.5 | 'A' | `りんご: 5個` |
| 7 | `printf("値段: %.1f円\n", price);` | 5 | 120.5 | 'A' | `値段: 120.5円` |
| 8 | `printf("成績: %c\n", grade);` | 5 | 120.5 | 'A' | `成績: A` |
| 9 | `return 0;` | — | — | — | (プログラム終了) |

**最終出力:**
```

値段: 120.5円
成績: A
```

## 6. 視覚的メモリ図

プログラム実行中、変数はメモリ上に次のように配置されます：

```
メモリ（RAM）のイメージ
┌────────┬──────────────┬─────────┐
│ アドレス│  変数名      │  値     │
├────────┼──────────────┼─────────┤
│ 0x1000 │  apples(int) │    5    │  ← 4バイト占有
│ 0x1004 │  price(dbl)  │  120.5  │  ← 8バイト占有
│ 0x100C │  grade(char) │   'A'   │  ← 1バイト占有
└────────┴──────────────┴─────────┘
 ↑アドレスはコンパイラが自動管理（気にしなくてOK）

「箱」のイメージ（大きさが違う）:
 apples           price                  grade
┌────────┐    ┌────────────────┐    ┌──┐
│   5    │    │     120.5      │    │A │
└────────┘    └────────────────┘    └──┘
 int(4B)           double(8B)        char(1B)
```

## 7. よくある間違い

### 間違い1: scanf で & を忘れる

```c
// ❌ 間違い: プログラムが異常終了する可能性
int n;
scanf("%d", n);

// ✅ 正しい: & を必ず付ける
int n;
scanf("%d", &n);
```

**理由:** `&n` は「n の場所（アドレス）」を意味します。`&` なしでは「n の値（ゴミ値）」をアドレスとして解釈し、無効なメモリに書き込んでしまいます。

---

### 間違い2: double に %d を使う

```c
double x = 3.14;
// ❌ 間違い: 出力が化ける
printf("%d\n", x);

// ✅ 正しい: double には %f か %.Nf
printf("%f\n", x);    // → 3.140000
printf("%.2f\n", x);  // → 3.14
```

---

### 間違い3: 整数除算の落とし穴

```c
// ❌ 間違い: 1.5 を期待したのに 1 になる
int a = 3, b = 2;
double result = a / b;   // 右辺が int/int → 1（整数除算）
printf("%.1f\n", result); // → 1.0

// ✅ 正しい: どちらかを double にする
double result = (double)a / b;   // キャスト
printf("%.1f\n", result); // → 1.5
```

**ルール:** `int / int` は常に整数結果（小数切り捨て）になる。

---

### 間違い4: 未初期化変数を使う

```c
// ❌ 間違い: x の値は「ゴミ」（不定）
int x;
printf("%d\n", x);  // 何が出るか分からない（例: -1094795585）

// ✅ 正しい: 宣言と同時に初期化
int x = 0;
printf("%d\n", x);  // → 0
```

---

### 間違い5: ' と " を混同する

```c
char c;
// ❌ 間違い: 文字列（"A"）を char に代入しようとしている
c = "A";   // コンパイルエラー

// ✅ 正しい: 1文字はシングルクォート
c = 'A';   // OK

// 文字列はダブルクォート（char 配列が必要）
char s[] = "hello";  // OK
```

## 9. 例題（ワーク済み）

### 例題1: 2つの整数を入力して合計と平均を表示

**問題:** 2つの整数を入力し、合計と平均（小数点以下1桁）を表示するプログラムを書け。

**解答:**
```c
#include <stdio.h>

int main() {
    int a, b;
    
    printf("1つ目の整数: ");
    scanf("%d", &a);
    
    printf("2つ目の整数: ");
    scanf("%d", &b);
    
    int sum = a + b;
    double avg = (double)(a + b) / 2;  // キャストで実数除算
    
    printf("合計: %d\n", sum);
    printf("平均: %.1f\n", avg);
    
    return 0;
}
```

**実行例（入力: 7 と 3）:**
```
1つ目の整数: 7
2つ目の整数: 3
合計: 10
平均: 5.0
```

**ポイント:** `(double)(a + b) / 2` の `(double)` がキャスト。これがないと `(7+3)/2 = 5`（整数除算）になる。

---

### 例題2: 氏名と年齢を入力して表示

```c
#include <stdio.h>

int main() {
    char name[50];  // 文字列は char の配列（L05で詳しく学ぶ）
    int  age;
    
    printf("名前: ");
    scanf("%s", name);   // 文字列は & 不要（配列だから）
    
    printf("年齢: ");
    scanf("%d", &age);   // int は & 必要
    
    printf("こんにちは、%sさん（%d歳）\n", name, age);
    
    return 0;
}
```

**実行例:**
```
名前: Takao
年齢: 22
こんにちは、Takaoさん（22歳）
```

### 例題3: 型変換の確認

```c
#include <stdio.h>

int main() {
    int    a = 10, b = 3;
    double result1 = a / b;              // 整数除算 → 3.0
    double result2 = (double)a / b;      // キャスト → 3.333...
    double result3 = (double)(a / b);    // 整数除算後キャスト → 3.0
    
    printf("a / b         = %.3f\n", result1); // → 3.000
    printf("(double)a / b = %.3f\n", result2); // → 3.333
    printf("(double)(a/b) = %.3f\n", result3); // → 3.000
    
    return 0;
}
```

**ポイント:** キャストの位置が結果を決定する。`(double)a / b` と `(double)(a/b)` は全く異なる。

## 10. 新潟大学試験との関連

**出題頻度: Tier 1 ★ (毎年必出)**

C言語の変数・型・printf は3年間すべての試験に登場しています。

### 典型的な出題パターン

**パターン A: 出力結果を答えよ**（最頻出）

```c
int a = 5, b = 2;
printf("%d\n", a / b);
printf("%.1f\n", (double)a / b);
```

→ 出力は `2` と `2.5`。整数除算とキャストの理解が必要。

**パターン B: 書式指定子の穴埋め**

```c
double x = 3.14;
printf("___\n", x);   // 3.14 を表示するには？
```

→ `"%.2f"` が正解。`%d` は誤り（型不一致）。

**パターン C: scanf の & の有無**

「次のプログラムのバグを修正せよ」という形で `&` 忘れが出題されることがある。

### 試験対策チェックリスト

- [ ] int, double, char の使い分けができる
- [ ] %d, %f, %.Nf, %c, %s を正しく使い分けられる
- [ ] 整数除算（3/2=1）を理解している
- [ ] キャスト (double)a で実数除算できる
- [ ] scanf に & が必要な理由を説明できる

## ミニクイズ 解答

**A1.** `7`（int 型 x の値がそのまま出力）

**A2.** `1`（int / int = 整数除算、小数点以下切り捨て）

**A3.** `n` の前に `&` が必要。正しくは `scanf("%d", &n);`

**A4.** `printf("%.2f\n", price);`

**A5.** `(b) my_score`
- (a) 数字始まりは不可
- (c) ハイフン `-` は演算子として解釈されるため不可
- (d) `int` はC言語の予約語（キーワード）なので変数名に使えない
