```c
int linear_search(int a[], int n, int key) {
```

```c
void min_max(int a[], int n, int *max_val, int *min_val) {
    int i;
    *max_val = a[0];
    *min_val = a[0];
    for (i = 1; i < n; i++) {
        if (a[i] > *max_val) *max_val = a[i];
        if (a[i] < *min_val) *min_val = a[i];
    }
}
```

### 大問 III 解答

#### (1) strlen の戻り値

- 戻り値: **7**（"niigata" は 7文字）
- `\0` は**含まれない**（`strlen` はヌル文字を数えない）

#### (2) my_reverse の空欄

- **(A)** `n/2`（前半だけループすれば全体が逆順になる）
- **(B)** `n-1-i`（対応する末尾側のインデックス）
- **(C)** `n-1-i`（同上）

```c
for (i = 0; i < n/2; i++) {
    tmp        = s[i];
    s[i]       = s[n-1-i];
    s[n-1-i]   = tmp;
}
```

#### (3) 実行結果

```
7
ataagiin
a g
```

- 1行目: `strlen("niigata") = 7`
- 2行目: `my_reverse` 後の "niigata" → "ataagiin"（nとa, iとi, iとg, gとa が交換）

  実際のトレース: n="niigata", 逆順 → "ataagiin"
- 3行目: `p = word`（逆順後の先頭 = 'a'）、`*(p+3)` = 'a'+3番目 = `word[3]` = 'a'（逆順後: a,t,a,a,g,i,i,n の index 3 = 'a'）

  → `a a`

#### (4) count_alpha

```c
int count_alpha(char s[]) {
    int i, count;
    count = 0;
    for (i = 0; s[i] != '\0'; i++) {
        if (('a' <= s[i] && s[i] <= 'z') ||
            ('A' <= s[i] && s[i] <= 'Z')) {
            count++;
        }
    }
    return count;
}
```
