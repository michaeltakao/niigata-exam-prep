**目標:** 再帰関数の実行結果を変数トレース表で追える。

- [ ] **Step 2** 以下を手でトレースする（紙に書く）
  ```c
  int fact(int n) {
      if (n <= 1) return 1;
      return n * fact(n - 1);
  }
  ```
  `fact(4)` の計算過程を木の形で書く:
  ```
  fact(4) = 4 × fact(3)
              = 4 × 3 × fact(2)
                    = 4 × 3 × 2 × fact(1)
                              = 4 × 3 × 2 × 1 = 24
  ```

## Day 10 — ポインタ（基礎のみ）

**目標:** `&` と `*` の意味がわかる。swap 関数が読める。

- [ ] **Step 1** [L07 教科書を読む](02_textbook/L08_pointers.md)（全部）
- [ ] **Step 2** メモリ図を紙に書く
  ```
  アドレス | 値
  [1000]  |  5   ← int x = 5;
  [1004]  | 1000 ← int *p = &x;
  ```
  - `&x` = x のアドレス = 1000
  - `*p` = p が指す場所の値 = 5
- [ ] **Step 3** 以下を手で追う
  ```c
  int x = 10;
  int *p = &x;
  *p = 20;
  printf("%d", x);  // → 何が出力される？
  ```
  （答え: 20　なぜなら *p = p が指す場所 = x の中身を変えた）
- [ ] **Step 4** [L07 ドリル](03_drills/L7_pointers.md) — Easy E1〜E4
- [ ] **Step 5** 答えを確認、間違いを直す

> 📌 詰まったら → [ポインタ補強](10_remediation/pointers.md)

**今日の終了チェック:**
- `p` と `*p` と `&x` の違いが言える
- `*p = 20;` が `x` の値を変えることが説明できる → 完了 ✅
