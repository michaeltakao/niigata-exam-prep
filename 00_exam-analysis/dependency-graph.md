## 暫定チェーン（一般的なC言語順序）

```
L01 変数・型
  └─→ L02 演算子・式
        └─→ L03 条件分岐 (if/else, switch)
              └─→ L04 ループ (for, while, do-while)
                    ├─→ L05 配列
                    │     └─→ L06 文字列 (char[])
                    │           └─→ L10 アルゴリズム (探索・ソート)
                    └─→ L07 関数
                          └─→ L08 ポインタ
                                └─→ L09 構造体
                                      └─→ L11 再帰
```

過去問分析後、各ノードに「この過去問で必要」のラベルを追加する。
