## はじめに（最初に30秒だけ読む）

- **使い方**: 下の表で今日の日付を探す → その日のブロックへ進む → リンクをタップして教材を開く。

- **スキップ判断**: 構造体・ファイル I/O・ポインタ（応用）は**やらなくてよい**（新潟大学では出題実績が薄い）。

## 毎日のやること（Day 1〜7）

各ブロックは「教科書 → ドリル →（苦手なら）補強 → スコア記録」の順です。
ドリルが終わったら必ず点数を記録 → [スコア記録シート](06_progress/student-progress.md)。

### Day 1（06-25）— 変数・型 ＋ 演算子

- 教科書: [L01 変数・データ型](02_textbook/L01_variables-types.md) / [L02 演算子・式](02_textbook/L02_operators-expressions.md)
- ドリル: [L1 変数ドリル](03_drills/L1_variables.md) / [L2 式ドリル](03_drills/L2_expressions.md)
- **最低ここだけ**: `int` / `float` / `char` を何も見ずに宣言できる。`3 + 4 * 2 = 11` の理由を言える。
- **合格ライン**: L1・L2 ドリルで正答 **70% 以上**。
- スコア記録 → [student-progress.md](06_progress/student-progress.md)

### Day 2（06-26）— 条件分岐 ＋ ループ入門

- 教科書: [L03 条件分岐](02_textbook/L03_conditions.md) / [L04 ループ](02_textbook/L04_loops.md)（前半だけでOK）
- ドリル: [L3 条件分岐ドリル](03_drills/L3_conditions.md) / [L4 ループドリル](03_drills/L4_loops.md)（前半）

- **最低ここだけ**: `if / else if / else` が書ける。`for` ループを手でトレースできる。
- **合格ライン**: L3 ドリルで正答 **70% 以上**、L4 前半を1問でもトレース完了。
- スコア記録 → [student-progress.md](06_progress/student-progress.md)

### Day 3 ★（06-27）— ループ仕上げ

- 教科書: [L04 ループ](02_textbook/L04_loops.md)（後半・ネストループ）
- ドリル: [L4 ループドリル](03_drills/L4_loops.md)（後半）

- **最低ここだけ**: 二重 `for` のトレース表が書ける。`while` ループを自力で書ける。
- **合格ライン**: L4 ドリル全体で正答 **70% 以上**。
- スコア記録 → [student-progress.md](06_progress/student-progress.md)

### Day 4 ★（06-28）— 配列 ＋ バブルソート

- **合格ライン**: L5 ドリルで正答 **65% 以上**。
- スコア記録 → [student-progress.md](06_progress/student-progress.md)

### Day 5 ★（06-29）— 関数（＋再帰さわり）

## よくある質問（FAQ）

**Q1. どこから始めればいい？**
このページの「今日は何日目？」表で今日の日付を探し、その日のブロックを上から順にやるだけです。考えすぎなくて大丈夫。

**Q2. 教科書を読んでも理解できない。**
例をそのままノートに写してください。「理解してから書く」ではなく「書きながら理解する」のが C 言語の学び方です。写したら、何も見ずに同じコードを書いてみる。

**Q3. ドリルの答えはどこ？**
各ドリルファイルの下の方に答えがあります。先に自分で解いてから答えを見てください。

**Q4. ループ／配列／関数が全然わからない。**
補強教材を開いてください → [ループ](10_remediation/loops.md) ／ [配列](10_remediation/arrays.md) ／ [関数](10_remediation/functions.md)。教科書より詳しい説明があります。

**Q5. 時間がなくて全部できそうにない。**
スキップしてよいものがあります：
- **スキップOK**: 構造体（L09）、ファイル I/O（L10）、ポインタの Medium / Hard（[L08](02_textbook/L08_pointers.md)・[L7ドリル](03_drills/L7_pointers.md)）
- **絶対にスキップしない**: ループ（L04）・配列（L05）・関数（L07）。この3つで試験の 65 点分です。

各日の「**最低ここだけ**」だけでも、2時間あれば核は回せます。

**Q6. 模擬試験 A で点が取れなかった。**
落ち込まなくて大丈夫。間違えた問題のトピックを書き出して、ループが弱ければ [L4ドリル](03_drills/L4_loops.md)、配列なら [L5ドリル](03_drills/L5_arrays.md)、関数なら [L6ドリル](03_drills/L6_functions.md) をやり直します。1問10分が目安。
