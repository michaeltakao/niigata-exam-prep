# L1 変数・型・printf/scanf ドリル

**使い方:** まず自力で解いてから解答を確認すること。理解度 0→3 で記録。

---

## ★ Easy（直接トレース）

### E1. 整数の出力

**問題:**
```c
#include <stdio.h>
int main() {
    int x = 42;
    printf("%d\n", x);
    return 0;
}
```
出力結果を書け。

💡 **ヒント:** `%d` は int 型の値を10進数で出力する。

✅ **解答:** `42`

📝 **解説:** `x` に 42 が代入されている。`printf` の `%d` は int の値をそのまま10進数で出力し、`\n` で改行する。

---

### E2. 負の整数

**問題:**
```c
#include <stdio.h>
int main() {
    int a = 10, b = 3;
    printf("%d\n", a - b - a);
    return 0;
}
```
出力結果を書け。

💡 **ヒント:** 左から順に計算する。

✅ **解答:** `-3`

📝 **解説:** `a - b - a = 10 - 3 - 10 = -3`。`%d` は負の数も符号付きで出力できる。

---

### E3. 複数変数

**問題:**
```c
#include <stdio.h>
int main() {
    int x = 1, y = 2, z = 3;
    printf("%d %d %d\n", z, y, x);
    return 0;
}
```
出力結果を書け。

💡 **ヒント:** printf の引数は左から順に `%d` に対応する。

✅ **解答:** `3 2 1`

📝 **解説:** 変数の宣言順ではなく、printf の引数に渡す順番で出力される。z=3, y=2, x=1 の順で渡しているので `3 2 1` となる。

---

### E4. double の出力

**問題:**
```c
#include <stdio.h>
int main() {
    double pi = 3.14159;
    printf("%f\n", pi);
    return 0;
}
```
出力結果を書け。

💡 **ヒント:** `%f` はデフォルトで小数点以下6桁を出力する。

✅ **解答:** `3.141590`

📝 **解説:** `%f` は小数点以下6桁で出力する。3.14159 は5桁なので末尾に 0 が補われて `3.141590` となる。

---

### E5. 小数点桁数の指定

**問題:**
```c
#include <stdio.h>
int main() {
    double x = 3.14159;
    printf("%.2f\n", x);
    return 0;
}
```
出力結果を書け。

💡 **ヒント:** `%.2f` は小数点以下2桁で四捨五入して出力する。

✅ **解答:** `3.14`

📝 **解説:** `%.2f` は小数点以下2桁に丸める。3.14159 → 3.14（3桁目は1なので切り捨て）。

### E6. char の出力

✅ **解答:** `A`

📝 **解説:** `char` はシングルクォートで囲んだ文字1文字を格納する。`%c` で文字として出力するので `A` が表示される。

### E8. 整数除算の落とし穴

**問題:**
```c
#include <stdio.h>
int main() {
    int result = 7 / 2;
    printf("%d\n", result);
    return 0;
}
```
出力結果を書け。

💡 **ヒント:** int / int は整数除算（余りを切り捨て）になる。

✅ **解答:** `3`

📝 **解説:** `7 / 2` は両辺が int なので整数除算となり、商の整数部分のみ（3.5 の小数部切り捨て → 3）が得られる。小数が必要なら `7.0 / 2` か `(double)7 / 2` と書く。

### E9. キャスト

**問題:**
```c
#include <stdio.h>
int main() {
    double result = (double)7 / 2;
    printf("%.1f\n", result);
    return 0;
}
```
出力結果を書け。

💡 **ヒント:** (double)7 により左辺が double になる。double / int → double。

✅ **解答:** `3.5`

📝 **解説:** `(double)7` により 7 が 7.0 として扱われる。7.0 / 2 は double の除算になるので 3.5 となる。`%.1f` で小数点以下1桁表示。

---

### E10. 複数型の出力

**問題:**
```c
#include <stdio.h>
int main() {
    int n = 10;
    double x = 2.5;
    char c = 'Z';
    printf("%d %.1f %c\n", n, x, c);
    return 0;
}
```
出力結果を書け。

💡 **ヒント:** 各 `%` 指定子に対応する引数が順に使われる。

✅ **解答:** `10 2.5 Z`

📝 **解説:** 3つの変数が左から順に `%d`, `%.1f`, `%c` に対応する。スペース区切りで `10 2.5 Z` が出力される。

## ★★ Medium（変形・穴埋め）

### M1. scanf の & 忘れ

**問題:** 次のコードは「整数を入力して2倍にして出力」するプログラムである。空欄 (A) を埋めよ。

```c
#include <stdio.h>
int main() {
    int n;
    scanf("%d", (A));
    printf("%d\n", n * 2);
    return 0;
}
```

💡 **ヒント:** `scanf` は変数のアドレスを受け取る必要がある。

✅ **解答:** `(A)` = `&n`

📝 **解説:** `scanf` は入力値を変数に書き込むため、変数のアドレス（`&n`）を渡す必要がある。`&` を忘れると未定義動作（クラッシュや誤動作）になる。

---

### M2. scanf の書式指定子

**問題:** 次のコードの空欄 (A) を埋めよ。入力 `3.14` で `x = 3.14` と出力する。

```c
#include <stdio.h>
int main() {
    double x;
    scanf((A), &x);
    printf("x = %.2f\n", x);
    return 0;
}
```

💡 **ヒント:** double を scanf で読む書式指定子は `%lf`（小文字 l + f）。

✅ **解答:** `(A)` = `"%lf"`

📝 **解説:** `scanf` で double を読むには `%lf` を使う（`printf` は `%f` で OK だが `scanf` は `%lf` が必要）。この区別は試験でよく問われる。

### M3. 変数の初期化なし

**問題:** 次のコードを実行したとき、出力は何か。理由も説明せよ。

```c
#include <stdio.h>
int main() {
    int x;
    printf("%d\n", x);
    return 0;
}
```

💡 **ヒント:** 宣言だけして代入していない変数の値は？

✅ **解答:** 不定値（実行のたびに変わる可能性がある）。出力は予測不可能。

📝 **解説:** C言語では初期化していないローカル変数の値は「不定」。メモリ上の残りかすの値がそのまま使われる。`int x = 0;` のように必ず初期化する習慣が重要。

### M4. キャストの順序

**問題:** 次の3つの式の値を求めよ。

```c
int a = 5, b = 2;
(1) (double)(a / b)
(2) (double)a / b
(3) a / (double)b
```

💡 **ヒント:** (1) は整数除算の結果をキャスト、(2)(3) は除算前にどちらかが double になる。

✅ **解答:**  
(1) `2.0`（先に 5/2=2 の整数除算、その後 double にキャスト）  
(2) `2.5`（5 が 5.0 になってから除算）  
(3) `2.5`（b が 2.0 になってから除算）

📝 **解説:** (1) はカッコの中で先に整数除算が行われ、結果 2 を double にキャストするので 2.0。(2)(3) は除算前に片方が double になるため浮動小数点除算になり 2.5 となる。

### M5. 型変換の連鎖

**問題:** 次の出力を答えよ。

```c
#include <stdio.h>
int main() {
    int x = 10;
    double y = x / 3;
    printf("%.2f\n", y);
    return 0;
}
```

💡 **ヒント:** 右辺の `x / 3` の型は何か？

✅ **解答:** `3.00`

📝 **解説:** `x / 3` は int / int = 3（整数除算）。その後 3 が double の y に代入されるとき 3.0 になる。最終的に `%.2f` で `3.00` と出力される。

### M6. オーバーフローの概念

**問題:** int 型の最大値を 2147483647 とする。次の出力を予測せよ（処理系依存だが一般的な動作を答えよ）。

```c
#include <stdio.h>
#include <limits.h>
int main() {
    int x = INT_MAX;
    printf("%d\n", x + 1);
    return 0;
}
```

💡 **ヒント:** int の最大値に 1 を足すと何が起きるか？

✅ **解答:** `-2147483648`（負の最大値 = INT_MIN）

📝 **解説:** int が表現できる最大値 (2^31−1) を超えると「オーバーフロー」が発生し、2の補数表現でビットが折り返して最小値 (−2^31) になる。試験では「動作が未定義」と書けば正解になることも多い。

### M7. unsigned と signed

**問題:** 次の出力を答えよ。

```c
#include <stdio.h>
int main() {
    unsigned int u = 0;
    printf("%d\n", u - 1);
    return 0;
}
```

💡 **ヒント:** unsigned int に `%d` を使うとどうなる？

✅ **解答:** `-1`（環境によるが多くの場合）

📝 **解説:** unsigned 0 から 1 を引くとラップアラウンドで unsigned の最大値になるが、`%d`（signed 解釈）で印字すると −1 と表示される環境が多い。試験では「符号なし整数の減算でラップアラウンド」と書けばよい。

### M8. 書式指定子の穴埋め

**問題:** 次のコードで `12 3.14 A` と出力するための空欄を埋めよ。

```c
#include <stdio.h>
int main() {
    int n = 12;
    double x = 3.14;
    char c = 'A';
    printf("(A) (B) (C)\n", n, x, c);
    return 0;
}
```

💡 **ヒント:** int→`%d`, double→`%f` or `%.2f`, char→`%c`。

✅ **解答:** `(A)=%d`, `(B)=%.2f`, `(C)=%c`

📝 **解説:** 各型に対応する書式指定子を使う。`%.2f` とすると 3.14 が小数点以下2桁で出力される。`%f` だと `3.140000` になるので `.2` を指定して桁数を制限する。

### M10. 変数の上書き

**問題:** 次のコードの出力を書け。

```c
#include <stdio.h>
int main() {
    int x = 5;
    x = x + 3;
    x = x * 2;

    return 0;
}
```

💡 **ヒント:** 代入は上書きする。右辺を評価してから左辺に書き込む。

✅ **解答:** `16`

📝 **解説:** まず x = 5 + 3 = 8、次に x = 8 * 2 = 16。`=` は「比較」ではなく「代入（上書き）」なので右辺の計算結果が x に書き込まれる。

## ★★★ Hard（入試形式）

### H1. キャストの出力トレース

**問題:** 次のコードの出力をすべて書け。

```c
#include <stdio.h>
int main() {
    int a = 7, b = 2;
    printf("%d\n", a / b);
    printf("%.1f\n", (double)a / b);
    printf("%.1f\n", (double)(a / b));
    printf("%d\n", a % b);
    return 0;
}
```

💡 **ヒント:** 各行のキャストがいつ適用されるか注意。

✅ **解答:**
```
3
3.5
3.0
1
```

📝 **解説:** 1行目: 整数除算 7/2=3。2行目: a を double にキャストしてから除算→3.5。3行目: (a/b)=3 を後からキャスト→3.0。4行目: 余り 7%2=1。`(double)` の位置でまったく違う結果になることを理解する。

### H2. 型変換を含む多行出力

**問題:** 次のプログラムの出力をすべて書け。

```c
#include <stdio.h>
int main() {
    double x = 9.9;
    int n = (int)x;
    printf("%d\n", n);
    printf("%.1f\n", x - n);
    printf("%d\n", (int)(x + 0.5));
    return 0;
}
```

💡 **ヒント:** `(int)` は小数部を切り捨てる（四捨五入ではない）。

✅ **解答:**
```
9
0.9
10
```

📝 **解説:** `(int)9.9 = 9`（小数部切り捨て）。`x - n = 9.9 - 9 = 0.9`。`(int)(9.9 + 0.5) = (int)10.4 = 10`（+0.5してから切り捨てで四捨五入効果）。

### H3. 書式と型の不一致

**問題:** 次のコードの出力を答えよ（または問題点を指摘せよ）。

```c
#include <stdio.h>
int main() {
    int n = 65;
    printf("%c\n", n);
    printf("%d\n", 'A');
    return 0;
}
```

💡 **ヒント:** char と int は内部的には整数。ASCIIコード 65 は？

✅ **解答:**
```
A
65
```

📝 **解説:** `int` を `%c` で出力するとASCIIコードとして文字に変換される。65 = 'A'。逆に `'A'` を `%d` で出力するとASCIIコード 65 が出力される。C言語では `char` と `int` の変換が自然に行われる。

### H4. 変数宣言の穴埋め

**問題:** 次のコードが正しく動作するよう (A)〜(C) を埋めよ。
期待出力: `score=87 average=87.50 grade=B`

```c
#include <stdio.h>
int main() {
    (A) score = 87;
    (B) average = score / 2.0 + 43.5;
    (C) grade = 'B';
    printf("score=%d average=%.2f grade=%c\n", score, average, grade);
    return 0;
}
```

💡 **ヒント:** それぞれの型を printf の書式指定子から判断する。

✅ **解答:** `(A)=int`, `(B)=double`, `(C)=char`

📝 **解説:** `%d` → int, `%.2f` → double, `%c` → char。`score/2.0 + 43.5 = 87/2.0 + 43.5 = 43.5 + 43.5 = 87.0` なので average は double で格納する必要がある。

### H5. 複合代入と型

**問題:** 次のプログラムの出力をすべて書け。

```c
#include <stdio.h>
int main() {
    int x = 100;
    double y = 100;
    x /= 3;
    y /= 3;
    printf("%d\n", x);
    printf("%.4f\n", y);
    printf("%d\n", x * 3);
    printf("%.4f\n", y * 3);
    return 0;
}
```

💡 **ヒント:** int の `/=` と double の `/=` で異なる結果になる。

✅ **解答:**
```
33
33.3333
99
100.0000
```

📝 **解説:** `x /= 3` は整数除算で `x = 33`。`x * 3 = 99`（元の100に戻らない！）。`y /= 3` は浮動小数点除算で `y = 33.3333...`。`y * 3 ≈ 100.0000`（近似的に元に戻る）。

### H6. 入力を受け取る計算機

**問題:** 次の仕様のプログラムを完成させよ。  
仕様: 2つの整数 a, b を入力して、和・差・積・商（double）を出力する。

```c
#include <stdio.h>
int main() {
    // ここを埋めよ
}
```

入力例: `10 3`  
出力例:
```
sum=13
diff=7
product=30
quotient=3.33
```

💡 **ヒント:** quotient は `(double)a / b` で計算。

✅ **解答:**
```c
#include <stdio.h>
int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("sum=%d\n", a + b);
    printf("diff=%d\n", a - b);
    printf("product=%d\n", a * b);
    printf("quotient=%.2f\n", (double)a / b);
    return 0;
}
```

📝 **解説:** `scanf` で2つの int を読む。商だけ `(double)a / b` でキャストして浮動小数点除算にする。各計算結果を別々の printf で出力する。

### H7. 温度変換器

**問題:** 摂氏温度 C を入力して、華氏温度 F = C × 9/5 + 32 を出力するプログラムを書け。小数点以下1桁で出力。

入力例: `100`  
出力例: `212.0 F`

💡 **ヒント:** `9/5` は整数除算で 1 になる！`9.0/5` または `(double)9/5` を使う。

✅ **解答:**
```c
#include <stdio.h>
int main() {
    double celsius;
    scanf("%lf", &celsius);
    double fahrenheit = celsius * 9.0 / 5 + 32;
    printf("%.1f F\n", fahrenheit);
    return 0;
}
```

📝 **解説:** `celsius` は小数を扱うため double。`9/5` は整数除算で1になってしまうので `9.0/5` と書く。scanf で double を読む場合は `%lf` が必要。

### H8. BMI 計算器

**問題:** 身長(m)と体重(kg)を入力して BMI = 体重 / 身長² を計算して出力せよ。小数点以下2桁。

入力例: `1.70 65.0`  
出力例: `BMI=22.49`

💡 **ヒント:** 身長の二乗は `height * height`（`pow` 関数なしで書く）。

✅ **解答:**
```c
#include <stdio.h>
int main() {
    double height, weight;
    scanf("%lf %lf", &height, &weight);
    double bmi = weight / (height * height);
    printf("BMI=%.2f\n", bmi);
    return 0;
}
```

📝 **解説:** 入力はどちらも小数点を含む可能性があるので double。`height²` は `height * height` で計算する。BMI = 65.0 / (1.70 × 1.70) = 65.0 / 2.89 ≈ 22.49。

### H9. 入力値のスワップ

**問題:** 2つの整数を入力し、順番を入れ替えて出力するプログラムを書け。一時変数 `tmp` を使うこと。

入力例: `5 8`  
出力例: `8 5`

💡 **ヒント:** `tmp = a; a = b; b = tmp;` の3ステップで入れ替え。

✅ **解答:**
```c
#include <stdio.h>
int main() {
    int a, b, tmp;
    scanf("%d %d", &a, &b);
    tmp = a;
    a = b;
    b = tmp;
    printf("%d %d\n", a, b);
    return 0;
}
```

📝 **解説:** `a = b` と直接書くと元の a の値が失われる。先に `tmp = a` で保存してから入れ替える。この「3変数スワップ」はソートアルゴリズムの基本操作として試験頻出。

### H10. ← 入試形式 総合問題

**問題:** 次のプログラムについて (1)〜(4) に答えよ。

```c
#include <stdio.h>
int main() {
    int a = 10, b = 3;
    double x;
    
    x = a / b;
    printf("(1) x = %.2f\n", x);
    
    x = (double)a / b;
    printf("(2) x = %.2f\n", x);
    
    printf("(3) %d\n", a % b + b % a);
    
    a = a + b;
    b = a - b;
    a = a - b;
    printf("(4) a=%d b=%d\n", a, b);
    return 0;
}
```

(1) 1行目の出力を書け。  
(2) 2行目の出力を書け。  
(3) 3行目の出力を書け。  
(4) 4行目の出力を書け（スワップ後の値）。

💡 **ヒント:** (3) は 10%3 と 3%10 をそれぞれ計算。(4) は加減算でのスワップテクニック。

✅ **解答:**
```
(1) x = 3.00
(2) x = 3.33
(3) 4
(4) a=3 b=10
```
