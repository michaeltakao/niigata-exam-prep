# P06 角運動量と衝突

## 1. 直感的説明

回転運動の「運動量」が角運動量です。
力が働かなければ角運動量は保存されます（角運動量保存則）。

---

## 2. 角運動量

点 O まわりの角運動量:

::: formula
$$L = I\omega \quad\text{（剛体の場合）}$$
$$L = mvr \quad\text{（質点の場合。$r$: Oからの距離）}$$
:::

---

## 3. 慣性モーメント

| 形状 | 軸 | $I$ |
|---|---|---|
| 細い棒 (長さ $L$, 質量 $M$) | 一端を通る | $\frac{1}{3}ML^2$ |
| 細い棒 | 中心を通る | $\frac{1}{12}ML^2$ |
| 円板 (半径 $R$) | 中心軸 | $\frac{1}{2}MR^2$ |
| 質点 | 距離 $r$ | $mr^2$ |

::: formula
**平行軸の定理**:
$$I = I_G + Md^2 \quad\text{（$I_G$: 重心まわり, $d$: 平行移動距離）}$$
:::

---

## 4. 角運動量保存則

外力のトルクがゼロのとき:

::: formula
$$L_{\text{before}} = L_{\text{after}}$$
:::

衝突問題では「衝突の瞬間は外力の衝撃が大きいが、衝突時間が極めて短いため角運動量保存が成立」

---

## 5. 例題 — 棒への衝突 (新潟大学 R7 問題II)

::: example
長さ $L$, 質量 $M$ の棒が端 O を固定軸として静止。
端 P に質量 $m$, 速度 $v$ の小球が垂直に衝突してくっついた。

**衝突前の角運動量** (O まわり):
$$L_0 = mvL \quad\text{（棒は静止なので小球分のみ）}$$

**衝突後の慣性モーメント**:
$$I = \frac{1}{3}ML^2 + mL^2 = \left(\frac{M}{3}+m\right)L^2$$

**角速度** (角運動量保存):
$$mvL = I\omega \Rightarrow \omega = \frac{mvL}{\left(\frac{M}{3}+m\right)L^2} = \frac{mv}{\left(\frac{M}{3}+m\right)L}$$

**失われたエネルギー**:

衝突前 KE: $\frac{1}{2}mv^2$

衝突後 KE: $\frac{1}{2}I\omega^2 = \frac{(mvL)^2}{2I} = \frac{m^2v^2}{2\left(\frac{M}{3}+m\right)}$

$$\Delta E = \frac{1}{2}mv^2 - \frac{m^2v^2}{2\left(\frac{M}{3}+m\right)} = \frac{mv^2}{2}\cdot\frac{M/3}{M/3+m} = \frac{Mmv^2}{6(M/3+m)}$$
:::

---

## 6. よくある間違い

::: warning
- **慣性モーメントの合成**: 棒+質点は $I_棒 + mr^2$ の和（$r=L$ で小球が端なら $mL^2$）
- **衝突はエネルギー保存ではない**: 完全非弾性衝突はエネルギーが失われる（角運動量保存のみ）
- **棒の一端固定**: 固定端まわりの $I_棒 = \frac{1}{3}ML^2$ (中心まわりの $\frac{1}{12}ML^2$ ではない)
:::

---

## 7. 新潟大学での出題パターン

::: examtip
- **R7 問題II**: 角運動量保存→慣性モーメント→角速度→エネルギー損失 の4小問
- **必要な知識**: 棒の慣性モーメント ($\frac{1}{3}ML^2$) は問題に与えられる場合が多い
:::
