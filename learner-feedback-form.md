# Learner Feedback Form

> Collect qualitative feedback per session or weekly.  
> Copy the template block below and paste it at the bottom of this file for each new session.  
> Running session logs go in `06_progress/student-progress.md` — this file holds only feedback.

---

## How to Use

1. At the end of a study session (or once per week), copy everything between the `---` markers below.
2. Paste it after the most recent entry and fill it in.
3. Be honest — this form exists to surface friction the score sheet misses.
