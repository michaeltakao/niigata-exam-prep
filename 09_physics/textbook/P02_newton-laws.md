# P02 ニュートンの法則

## 1. 直感的説明

「力 = 質量 × 加速度」。これが全ての力学の出発点です。
自由体図（FBD）で物体に働く全ての力を書き出してから方程式を立てます。

---

## 2. ニュートンの3法則

::: formula
**第1法則 (慣性の法則)**: 力が働かなければ物体は等速直線運動を続ける

**第2法則 (運動の法則)**:
$$\mathbf{F} = m\mathbf{a}$$

**第3法則 (作用・反作用)**: AがBに力 $\mathbf{F}$ を及ぼすと、BはAに $-\mathbf{F}$ を及ぼす
:::

---

## 3. 力の分解

力は向きを持つベクトル。水平成分と鉛直成分に分解して考える。

::: formula
大きさ $F$、角度 $\theta$ (水平からの角度):

$$F_x = F\cos\theta, \quad F_y = F\sin\theta$$
:::

---

## 4. 自由体図 (FBD) の描き方

**手順**:
1. 対象物体を孤立させて描く
2. 重力 $mg$ (常に真下)
3. 接触力 (法線力 $N$: 接触面に垂直、摩擦力 $f$: 接触面に沿う)
4. ひもの張力 $T$ (ひもに沿って引く方向)
5. 外力 $F$ (問題で与えられた力)

---

## 5. 運動方程式の立て方

::: formula
**手順**:
1. FBDを描く
2. 正方向を定める
3. 各方向の力の合計 = $ma$ と書く
4. 方程式を解く
:::

### 例題 — 基本的な運動方程式

::: example
質量 $m$ の物体が床の上を力 $F$ で押されて加速している。動摩擦係数 $\mu_k$、重力加速度 $g$。

FBD:
- 重力: $mg$ (下)
- 法線力: $N$ (上)
- 摩擦: $\mu_k N$ (後ろ)
- 外力: $F$ (前)

$y$ 方向 (静止): $N - mg = 0 \Rightarrow N = mg$

$x$ 方向: $F - \mu_k mg = ma$

$$a = \frac{F - \mu_k mg}{m}$$
:::

---

## 6. よくある間違い

::: warning
- **法線力は鉛直方向ではない**: 斜面では鉛直ではなく斜面に垂直
- **摩擦力の向き**: 運動方向の逆（または静止しようとする力と逆）
- **作用・反作用の対**: 異なる物体間の力のペア（同一物体内の力は作用・反作用ではない）
:::

---

## 7. 新潟大学での出題パターン

::: examtip
- **R8・R7 全問共通**: 「運動方程式を立てよ」が必ず来る
- **斜面問題 (R8 問題II)**: 斜面方向と法線方向に分解が必要
- **まず FBD を描く習慣**: 試験本番でも FBD を答案に書くと部分点になる
:::
