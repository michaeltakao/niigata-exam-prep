## 出題パターンの特徴

1. **プログラムによって出題構成が異なる**:
   - 機械システム工学: 物理+数学 (大問2×2科目)
   - 人間支援感性科学: 数学+電気回路+プログラミング
2. **積分は毎年必出**: 定積分・面積は全4年で出題。R6では回転体の体積も追加
3. **R6 から新判明**: 積の微分 ($$x^3\cos2x$$)、置換積分、回転体体積が出題
4. **計算量が多い**: 途中式を省略せずに書く練習が必要

## R6 新出: 積分の重要トピック追加

### 回転体の体積

$$V = \pi\int_a^b [f(x)]^2\,dx$$

### 置換積分 ($$t=\cos x$$)

$$\int_0^{\pi/2}\cos^3x\sin x\,dx$$

$$t=\cos x$$, $$dt=-\sin x\,dx$$ で解く → M05 に追加済み

### 半角公式による極限

$$\lim_{\theta\to0}\frac{1-\cos2\theta}{\theta^2}=2 \quad\text{(R6 出題)}$$

## よく出る計算パターン（大問別）

人間支援感性科学プログラムの数学は **大問3構成** が標準。各大問の典型パターンを把握して計算演習の的を絞る。

### 大問 I — 線形代数

| 出題パターン | 典型例 | 対策教科書 |
|---|---|---|
| 行列式を求めよ | 3×3 余因子展開 | M02 |
| 逆行列を求めよ | 2×2 公式 or 掃き出し法 | M02 |
| $$A^n$$ を求めよ | 対角化 → $$PDP^{-1}$$ | M03 |
| 固有値・固有ベクトル | 特性方程式 $$\det(A-\lambda I)=0$$ | M03 |
| Im(f), Ker(f) の次元 | 線形写像の行列表現 → rank | M04 |

**計算手順 (A^n 求め方):**
1. 特性方程式 $$\det(A-\lambda I)=0$$ → 固有値 $$\lambda_1, \lambda_2$$
2. 固有ベクトル $$\mathbf{v}_1, \mathbf{v}_2$$ を求める
3. $$P=(\mathbf{v}_1,\mathbf{v}_2)$$, $$D=\mathrm{diag}(\lambda_1,\lambda_2)$$
4. $$A^n = PD^nP^{-1}$$

### 大問 II — 積分・微分方程式

| 出題パターン | 典型例 | 対策教科書 |
|---|---|---|
| 定積分 | $$\int_0^1 x e^x\,dx$$ （部分積分） | M05 |
| 面積 | $$y=x^2$$ と $$y=x$$ の囲む面積 | M05 |
| 回転体の体積 | $$V=\pi\int_a^b [f(x)]^2\,dx$$ | M05 |
| 置換積分 | $$t=\cos x$$ など | M05 |
| 1階 ODE | 変数分離法 | M07 |
| 2階線形 ODE | 特性方程式 → 一般解 + 特殊解 | M07 |

**積分の鉄則:**
- $$\int xe^x\,dx$$: **部分積分** ($$u=x$$, $$dv=e^x\,dx$$)
- $$\int \cos^3 x\,dx$$: **置換** ($$t=\sin x$$) または三角恒等式
- $$\int_0^\infty e^{-x}\,dx$$: **広義積分** ($$\lim_{b\to\infty}$$)

### 大問 III — ベクトル

| 出題パターン | 典型例 |
|---|---|
| 内積・なす角 | $$\mathbf{a}\cdot\mathbf{b} = \vert a\vert \vert b\vert \cos\theta$$ |
| 外積・面積・体積 | $$\vert \mathbf{a}\times\mathbf{b}\vert $$ = 平行四辺形面積 |
| 線形独立の判定 | $$\det([\mathbf{v}_1,\mathbf{v}_2,\mathbf{v}_3]) \neq 0$$ |

## 他大学との難易度比較

| 大学 | 傾向 | 難易度 | 注意点 |
|---|---|---|---|
| **新潟大学 人間支援** | 積分・線形代数・ODE、計算量多い | ★★★☆☆ | 部分点重視、途中式を丁寧に |
| 長岡技科大 | 固有値・対角化・ベクトル解析が頻出 | ★★★★☆ | 証明問題あり、論述力も必要 |
| 電気通信大 | フーリエ変換・複素関数も範囲 | ★★★★☆ | 応用数学の知識が必要 |
| 豊橋技科大 | 微積分・線形代数が中心 | ★★★☆☆ | 新潟大学と難易度が近い |

**Fact**: 新潟大学は「解ける問題で確実に得点する」形式。部分点が取りやすい記述式のため、**途中式を省略しない** ことが最重要。

## 残り時間別の推奨優先順位

### 1ヶ月以上ある場合

```
Week 1-2: M05 積分（毎年必出） + M02 行列基礎
Week 3:   M07 ODE + M01 ベクトル
Week 4:   M03 固有値・対角化 + M06 極限
Week 5-:  過去問 R6〜R8 を3周
```

### 2週間の場合

```
Day 1-4:  M05 積分（定積分・面積・置換・部分積分）
Day 5-8:  M02 行列（行列式・逆行列・A^n）
Day 9-11: M07 ODE（変数分離・2階線形）
Day 12-14: 過去問 直近2年分を集中演習
```

### 1週間の場合（直前対策）

```
Day 1-2:  積分 — M05 Tier 1 ドリルのみ
Day 3-4:  線形代数 — 行列式・逆行列・固有値（計算パターン暗記）
Day 5:    ODE — 変数分離法のみ
Day 6-7:  過去問 R8（最新年度）2回転
```

> **Opinion**: 直前期は「新しいトピックを増やす」より「既知トピックの計算速度を上げる」方が得点につながる。
