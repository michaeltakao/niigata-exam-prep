# P04 複数物体の連立運動方程式

## 1. 直感的説明

複数の物体がひもや接触でつながっているとき、各物体の運動方程式を立てて連立します。
「加速度が同じかどうか」「張力が共通かどうか」が鍵。

---

## 2. 解法の手順

::: formula
1. 各物体に FBD を描く
2. 正方向（各物体ごとに）を定める
3. 各物体の運動方程式を書く
4. 幾何学的な拘束条件を使う（ひもの長さ一定 → 加速度の関係）
5. 連立して $a$, $T$ を求める
:::

---

## 3. アトウッドマシン (新潟大学 R7 問題I)

### 設定

質量 $m_1$ と $m_2$ がひもで結ばれ、なめらかな滑車を通じて接続。

### 運動方程式

物体 $m_1$ (下に動く方向を正):
$$m_1g - T = m_1a \quad\cdots(1)$$

物体 $m_2$ (上に動く方向を正):
$$T - m_2g = m_2a \quad\cdots(2)$$

### 解

::: formula
(1)+(2): $(m_1-m_2)g = (m_1+m_2)a$

$$a = \frac{(m_1-m_2)g}{m_1+m_2}, \quad T = \frac{2m_1m_2g}{m_1+m_2}$$
:::

### 例題

::: example
$m_1=2\,\text{kg}$, $m_2=3\,\text{kg}$, $g=9.8\,\text{m/s}^2$

$$a=\frac{(2-3)\times9.8}{2+3}=-1.96\;\text{m/s}^2$$（$m_2$ が下がる）

$$T=\frac{2\times2\times3\times9.8}{5}=23.52\;\text{N}$$
:::

---

## 4. 斜面上の連結物体

### 設定

$m_1$ が斜面 (傾角 $\theta$、摩擦係数 $\mu_k$)、$m_2$ がぶら下がり。ひもと滑車で連結。

運動方程式:
$$m_1a = m_1g\sin\theta - \mu_k m_1g\cos\theta - T \quad\cdots(1)$$
$$m_2a = m_2g - T \quad\cdots(2)$$

---

## 5. よくある間違い

::: warning
- **正方向の設定**: 各物体で別々に設定してよいが、一貫性が必要（ひもで繋がった物体は加速度の大きさが等しい）
- **張力は両端で同じ**: なめらかな（質量無視の）ひもと滑車なら $T$ は共通
- **$m_1>m_2$ でないと $a>0$ にならない**: 結果の符号で「どちらが下がるか」を確認
:::

---

## 6. 新潟大学での出題パターン

::: examtip
- **R7 問題I**: アトウッドマシン（式の立て方→解く→数値代入→エネルギー確認）
- **連立の手法**: 加法で $T$ を消去して $a$ を求めるのが最速
:::
