## 出題パターンの特徴

1. **コード読解 + 穴埋め**: プログラムを見せて「何を出力するか」または「空欄を埋めよ」形式が多い
2. **小問3〜5問構成**: 難易度が段階的に上がる
3. **実行結果を書かせる**: 手でトレースする練習が必須
4. **バグ修正**: 意図的に入れたバグを見つけて修正させる問題あり

## よく出るコードパターン

### パターン1: 配列ソート (バブルソート)

```c
for (int i = 0; i < n-1; i++) {
    for (int j = 0; j < n-1-i; j++) {
        if (a[j] > a[j+1]) {
            int tmp = a[j]; a[j] = a[j+1]; a[j+1] = tmp;
        }
    }
}
```

### パターン2: 再帰 (階乗)

```c
int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}
```

### パターン3: ポインタと配列

```c
int a[] = {1, 2, 3};
int *p = a;
printf("%d", *(p + 1));  // 2
```

### パターン4: 文字列操作 (strlen / strcpy / strcmp)

```c
#include <string.h>
char s[] = "hello";
printf("%zu\n", strlen(s));   // 5 (null終端を含まない長さ)

char t[10];
strcpy(t, s);                  // t に "hello" をコピー

if (strcmp(s, "hello") == 0) printf("equal");  // 0=等しい
```

> **試験頻出**: `strlen` の戻り値は `size_t`（符号なし整数）。`int` に代入すると比較で意図しない結果になることがある。

### パターン5: 線形探索

```c
int linear_search(int a[], int n, int key) {
    for (int i = 0; i < n; i++) {
        if (a[i] == key) return i;   // 見つかったインデックスを返す
    }
    return -1;   // 見つからなかった場合
}
```

### パターン6: 再帰関数（フィボナッチ・階乗）

```c
// 階乗 (再帰)
int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}

// フィボナッチ数列 (再帰) — R8 出題
int fib(int n) {
    if (n <= 1) return 1;
    return fib(n - 1) + fib(n - 2);
}
```

> **手トレース必須**: 試験では `fib(4)` などの戻り値を求める問題が多い。木構造展開を紙に書く練習をすること。

### パターン7: 文字列の実装（手書き reversal / コピー）

```c
// 文字列逆順 (インデックス版)
void reverse_idx(char s[]) {
    int n = strlen(s);
    for (int i = 0; i < n / 2; i++) {
        char tmp = s[i];
        s[i] = s[n - 1 - i];
        s[n - 1 - i] = tmp;
    }
}

// 文字列逆順 (ポインタ版)
void reverse_ptr(char *s) {
    char *left = s;
    char *right = s + strlen(s) - 1;
    while (left < right) {
        char tmp = *left;
        *left++ = *right;
        *right-- = tmp;
    }
}
```

### よくある失点パターン

| ミス | 具体例 | 対策 |
|---|---|---|
| **セミコロン忘れ** | `int a = 0` → コンパイルエラー | 文末確認を習慣化 |
| **配列の範囲外アクセス** | `int a[5]; a[5] = 1;` → 未定義動作 | インデックスは 0〜n-1 |
| **ポインタの `*` 忘れ** | `printf("%d", p)` → アドレスが出力 | `*p` で値にアクセス |
| **scanf の `&` 忘れ** | `scanf("%d", n)` → セグメンテーション違反 | `&n` と書く |
| **整数除算** | `3 / 2 = 1`（`1.5` にならない） | 片方を `3.0` か `(double)3` に |
| **`==` と `=` の混同** | `if (x = 0)` → 常に偽になる | `if (x == 0)` と2つ書く |
| **文字列比較に `==`** | `if (s == "hello")` → アドレス比較 | `strcmp(s, "hello") == 0` を使う |
